# وزير الفصيد - منصة الشعر النبطي الأصيل

منصة عصرية ومتطورة مخصصة للشعر النبطي الأصيل، توفر تجربة استماع ممتازة لأفضل القصائد والأصوات.

## 🌙 الميزات الرئيسية

- ✨ **تصميم حديث وفاخر** - واجهة داكنة بلمسات ذهبية أنيقة
- 🎵 **مشغل صوت متقدم** - لتجربة استماع احترافية وسلسة
- 📚 **تصنيفات منظمة** - حب، شرف، طبيعة، ذكريات، حكمة، احتفالات
- 👥 **قراء محترفون** - أفضل المقرئين والشعراء في المجال
- 📱 **تجاوب كامل** - يعمل على جميع الأجهزة والشاشات
- 🌐 **دعم RTL كامل** - تجربة عربية أصيلة متقنة
- ⚡ **سرعة عالية** - موقع محسّن للأداء والسرعة

## 🚀 البدء السريع

### المتطلبات
- Node.js 18+ و npm أو yarn

### التثبيت والتشغيل المحلي

```bash
# تثبيت المتطلبات
npm install

# تشغيل بيئة التطوير
npm run dev
```

الموقع متاح على `http://localhost:3000`

## 🌐 النشر المباشر على Vercel

### الخطوات:
1. ادفع المشروع إلى GitHub
2. اذهب إلى vercel.com
3. انقر "New Project"
4. اختر مستودع GitHub
5. اضغط "Deploy"

**سيكون لديك موقع مباشر في دقائق!**

## 📁 هيكل المشروع

```
src/
├── app/              # صفحات التطبيق
├── components/       # المكونات المعاد استخدامها
└── data/            # بيانات القصائد والمقرئين
```

## 🔧 التكنولوجيا المستخدمة

- Next.js 14 (React Framework)
- TypeScript (لكود آمن)
- Tailwind CSS (تصميم رائع)
- RTL Support الكامل
