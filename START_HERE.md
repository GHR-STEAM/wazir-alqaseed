## 🚀 Three Ways to Deploy

### Option 1: Vercel (Easiest - 3 Steps)
```
1. Push to GitHub
2. Import on Vercel.com
3. Deploy!
```
**Result:** Live URL in 2 minutes

### Option 2: Your Server
```
1. npm install && npm run build
2. Upload .next/ + public/ folders
3. Run: npm start
```

### Option 3: Static Hosting
```
1. Edit next.config.js (set output: 'export')
2. npm run build
3. Upload out/ folder to any static host
```

## 📝 Customization

### Add Your Content
Edit `src/data/poems.ts`:
```typescript
{
  id: 'your-poem',
  title: 'القصيدة الجديدة',
  poet: 'الشاعر',
  category: 'love',
  text: 'نص القصيدة',
  reciters: [{...}]
}
```

### Add Audio Files
1. Put MP3 files in `public/audio/`
2. Reference in data: `audioUrl: '/audio/your-file.mp3'`

### Change Design
Edit `tailwind.config.js`:
- `colors.primary` → Dark backgrounds
- `colors.accent` → Gold accents
