## 📋 قائمة الاختبارات

### ✅ اختبار 1: Check-CurrentSettings.ps1

**الوصف:** التحقق من أن السكريبت يعرض الإعدادات الحالية بشكل صحيح

**خطوات الاختبار:**
```powershell
cd "C:\Users\$env:USERNAME\Claude\Projects\ذيبان"
Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process
.\Check-CurrentSettings.ps1
```

**النتائج المتوقعة:**
- [ ] عرض الدقة الحالية (مثال: 1920x1080)
- [ ] عرض معدل التحديث الحالي (مثال: 60 Hz)
- [ ] عرض خطة الطاقة النشطة (High Performance أو Balanced)
- [ ] عرض معلومات GPU (اسم البطاقة + الذاكرة)
- [ ] عرض معلومات النظام (Windows, CPU, RAM)
- [ ] لا توجد رسائل خطأ

**الاختبار ناجح إذا:** تم عرض جميع المعلومات بدون أخطاء ✅

---

### ✅ اختبار 2: Get-SupportedRefreshRates.ps1

**الوصف:** التحقق من أن السكريبت يعثر على الترددات المدعومة

**خطوات الاختبار:**
```powershell
.\Get-SupportedRefreshRates.ps1
```

**النتائج المتوقعة:**
- [ ] عرض الدقة الحالية
- [ ] عرض معدل التحديث الحالي
- [ ] قائمة بجميع الترددات المدعومة (60Hz, 75Hz, 120Hz, 144Hz, إلخ)
- [ ] تمييز التردد الحالي بـ "الحالي 🟢"
- [ ] عرض أعلى تردد مدعوم
- [ ] توصيات الاستخدام (ألعاب/عمل)
- [ ] لا توجد رسائل خطأ

**الاختبار ناجح إذا:** تم عرض قائمة بالترددات بدون أخطاء ✅

---

### ✅ اختبار 3: Set-Resolution.ps1 (مع الخيارات)

**الوصف:** التحقق من أن السكريبت يطبّق الدقة ومعدل التحديث بشكل صحيح

#### الاختبار 3.1: الخيار الافتراضي
```powershell
.\Set-Resolution.ps1
# اختر: 1 (لتخطي الاختيار الافتراضي)
# اختر: 4 (لتخطي اختيار التردد الافتراضي)
```

**النتائج المتوقعة:**
- [ ] عرض الدقة الحالية ومعدل التحديث الحالي
- [ ] عرض قائمة الخيارات للدقة (1-5)
- [ ] عرض قائمة الخيارات للتردد (1-7)
- [ ] رسالة نجاح "✅ تم تطبيق الإعدادات بنجاح"
- [ ] عرض الإعدادات الجديدة المطبقة

**الاختبار ناجح إذا:** تم تطبيق 1920x1080 @ 144Hz بنجاح ✅

#### الاختبار 3.2: خيار مخصص
```powershell
.\Set-Resolution.ps1
# اختر: 5 (مخصص)
# أدخل: 1920x1080
# اختر: 7 (مخصص)
# أدخل: 60
```

**النتائج المتوقعة:**
- [ ] قبول القيم المخصصة
- [ ] تطبيق الدقة والتردد المخصص
- [ ] رسالة نجاح

**الاختبار ناجح إذا:** تم تطبيق القيم المخصصة بنجاح ✅

---

### ✅ اختبار 4: Set-PerformanceMode.ps1

**الوصف:** التحقق من أن السكريبت يفعّل وضع الأداء الأقصى

**خطوات الاختبار:**
```powershell
.\Set-PerformanceMode.ps1
```

**النتائج المتوقعة:**
- [ ] رسالة "✅ تم تعيين High Performance" (أو رسالة مشابهة)
- [ ] رسائل نجاح لـ:
  - [ ] Monitor Timeout
  - [ ] Disk Timeout
  - [ ] USB Selective Suspend
  - [ ] PCI Express Power Management
  - [ ] Hard disk sleep
- [ ] عرض خطة الطاقة النشطة الحالية (High Performance)
- [ ] ملخص الإعدادات المطبقة

**الاختبار ناجح إذا:** تم تفعيل High Performance بدون أخطاء ✅

---

### ✅ اختبار 5: Reset-DisplaySettings.ps1

**الوصف:** التحقق من أن السكريبت يعيد الإعدادات الافتراضية

**خطوات الاختبار:**
```powershell
.\Reset-DisplaySettings.ps1
```

**النتائج المتوقعة:**
- [ ] رسالة "✅ تم تعيين Balanced" (أو رسالة مشابهة)
- [ ] رسائل نجاح لـ:
  - [ ] Monitor Timeout (10 دقائق)
  - [ ] Disk Timeout (20 دقيقة)
  - [ ] USB Selective Suspend (مفعّل)
  - [ ] Display Resolution (1920x1080 @ 60Hz)
- [ ] عرض خطة الطاقة النشطة الحالية (Balanced)
- [ ] ملخص الإعدادات المُعادة

**الاختبار ناجح إذا:** تم إعادة تعيين جميع الإعدادات بدون أخطاء ✅

## 🔄 سيناريوهات الاختبار المتكاملة

### السيناريو 1: الاستخدام الأول (من الصفر)
```
1. Check-CurrentSettings.ps1       ← فحص الإعدادات الحالية
2. Get-SupportedRefreshRates.ps1   ← معرفة الخيارات المتاحة
3. Set-Resolution.ps1              ← تطبيق الدقة المطلوبة
4. Set-PerformanceMode.ps1         ← تفعيل الأداء
5. Check-CurrentSettings.ps1       ← التحقق من التطبيق الناجح
```

**النتيجة المتوقعة:** جميع الإعدادات مطبقة بنجاح ✅

---

### السيناريو 2: التبديل بين الألعاب والعمل
```
# للعمل:
Set-Resolution.ps1        → 1920x1080 @ 60Hz
Reset-DisplaySettings.ps1 → Balanced Mode

# للألعاب:
Set-Resolution.ps1        → 1920x1080 @ 144Hz
Set-PerformanceMode.ps1   → High Performance
```

**النتيجة المتوقعة:** تبديل سلس بين الإعدادات ✅

---

### السيناريو 3: حل المشاكل
```
1. Check-CurrentSettings.ps1       ← تشخيص المشكلة

3. Set-Resolution.ps1              ← محاولة إعدادات أخرى
4. Reset-DisplaySettings.ps1       ← العودة للافتراضي إذا لزم الأمر
```

**النتيجة المتوقعة:** حل المشكلة أو تقليل الضرر ✅
