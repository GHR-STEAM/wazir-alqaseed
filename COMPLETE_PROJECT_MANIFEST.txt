🛠️ BUILD INSTRUCTIONS
═══════════════════════════════════════════════════════════════════

1. INSTALL DEPENDENCIES
   Command: npm install
   Time: 2-3 minutes

2. DEVELOP LOCALLY (Optional)
   Command: npm run dev
   Open: http://localhost:3000

3. BUILD FOR PRODUCTION
   Command: npm run build
   Creates: .next/ folder (~50MB)

4. DEPLOY
   Choose your platform:

   A) Vercel (Easiest)
      • Push to GitHub
      • Import on vercel.com
      • Deploy with one click

   B) Your Server
      • Upload .next/ and public/
      • Run: npm start
      • Access on your domain

   C) Static Hosting
      • Set output: 'export' in next.config.js
      • Run: npm build (creates out/)
      • Upload out/ folder
